# Chrome Web Store — листинг FRKN Service Checker

## Основные поля

| Поле | Значение |
|------|----------|
| **Название** | FRKN Service Checker |
| **Короткое описание** | Проверяет доступность популярных сервисов, IP/VPN и скорость из браузера пользователя |
| **Категория** | Productivity (или Utilities) |
| **Язык** | Русский |
| **Контактная почта** | support@frkn.org |
| **Политика конфиденциальности** | https://status.frkn.org/privacy-en.html |
| **Домашняя страница** | https://status.frkn.org |

## Полное описание

FRKN Service Checker — это лёгкое браузерное расширение, которое помогает быстро понять, что происходит с вашим интернет-соединением:

- **Доступность сервисов** — проверяет доступность популярных сайтов и онлайн-сервисов: социальных сетей, видеоплатформ, мессенджеров, игровых сервисов, поисковых систем, государственных порталов и других ресурсов.
- **IP и VPN** — показывает ваш внешний IP, страну, провайдера, ASN и пытается определить, используется ли VPN или дата-центр.
- **Серверы FRKN** — загружает список нод FRKN и проверяет их доступность, включая API `api.frkn.org`.
- **Скорость соединения** — измеряет скорость загрузки через тестовые файлы.

Все проверки выполняются прямо в браузере пользователя. Расширение не собирает и не передаёт персональные данные.

## Разрешения

- `<all_urls>` — необходим для HEAD/GET-запросов к проверяемым сайтам. Без него невозможно отличить реальный сайт от провайдерской заглушки.

## Графика

- **Скриншоты** (обязательно): 1280×800 или 640×400, PNG/JPEG.
- **Маленькое промо** (опционально): 440×280.
- **Большое промо** (опционально): 920×680.
- **Marquee** (опционально): 1400×560.
- **Иконка** уже есть в архиве: 128×128 (`icons/icon128.png`).
