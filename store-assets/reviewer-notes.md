# Заметки для рецензента / Reviewer notes

## Назначение расширения

FRKN Service Checker помогает пользователям диагностировать доступность интернет-ресурсов и состояние их подключения:

- Проверяет доступность популярных сайтов (YouTube, X/Twitter, Instagram, Telegram, ChatGPT, Claude, Госуслуги, Яндекс, Roblox, Cloudflare, Google DNS) через HEAD/GET-запросы из браузера.
- Определяет внешний IP, страну, провайдера и ASN через публичные сервисы геолокации.
- Пытается определить, используется ли VPN/дата-центр.
- Загружает список серверов FRKN и проверяет их доступность, включая API `api.frkn.org`.
- Измеряет скорость интернет-соединения через загрузку тестовых файлов.

## Почему запрашивается `<all_urls>`

Расширению нужен доступ к разным доменам, потому что оно проверяет доступность множества сторонних сервисов напрямую из браузера пользователя. Без `host_permissions: ["<all_urls>"]` невозможно отправлять сетевые запросы к этим сайтам и корректно отличать реальный ответ от провайдерской блокировки/заглушки.

Расширение не внедряет скрипты на страницы, не модифицирует контент и не перехватывает трафик. Все запросы инициируются только по действию пользователя (открытие popup) или при нажатии кнопок «Обновить» / «Измерить скорость».

## Используемые внешние сервисы

- `ipapi.co/json/`, `ipwho.is/`, `ipinfo.io/json` — определение IP и геолокации.
- `api.frkn.org/nodes` — список нод FRKN и статус API.
- `status.frkn.org/speedtest/` — собственные файлы для замера скорости.
- `proof.ovh.net`, `cachefly.cachefly.net` — публичные зеркала для замера скорости (fallback).
- Проверяемые сайты из списка `SERVICES` в `background.js`.

## Данные и приватность

- Расширение не собирает, не хранит и не передаёт персональные данные пользователя.
- Результаты проверок отображаются только в интерфейсе popup и не сохраняются.
- Политика конфиденциальности: https://status.frkn.org/privacy.html
