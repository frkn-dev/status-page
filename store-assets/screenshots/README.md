# Скриншоты и промо-изображения

## Требования

### Chrome Web Store

- **Скриншоты**: 1280×800 или 640×400, PNG/JPEG, не более 12 шт.
- **Маленькое промо**: 440×280 (PNG/JPEG).
- **Большое промо**: 920×680 (PNG/JPEG).
- **Marquee**: 1400×560 (PNG/JPEG).

### Firefox Add-ons (AMO)

- **Скриншоты**: рекомендуется 1000×750 или 700×525, PNG/JPEG.
- **Промо-изображение**: 640×400 (PNG/JPEG).

## Рекомендуемые сцены для скриншотов

1. **Вкладка «Сервисы» после проверки**  
   Показывает список сервисов со статусами «Доступен» / «Недоступен» и задержкой.

2. **Вкладка «Ноды»**  
   Общий статус серверов FRKN и раскрытый список нод с инбаундами.

3. **Вкладка «IP/VPN»**  
   Внешний IP, страна, провайдер, ASN, статус VPN и доступность `api.frkn.org`.

4. **Вкладка «Скорость»**  
   Результат замера скорости в Мбит/с.

## Как сделать скриншоты popup

1. Установите расширение локально:
   - Chrome: `chrome://extensions/` → «Загрузить распакованное» → выбрать распакованный `frkn-service-checker-chrome.zip`.
   - Firefox: `about:debugging` → «Загрузить временное дополнение» → выбрать `manifest.json` из распакованного `frkn-service-checker-firefox.zip`.
2. Откройте popup, переключитесь на нужную вкладку и нажмите «Обновить».
3. Сделайте скриншот области popup.
4. Масштабируйте до нужного разрешения (например, 1280×800) с сохранением пропорций, добавив фон/обрамление при необходимости.

> Промо-изображения можно сделать в Figma/Photoshop на основе фирменных цветов FRKN: фон `#0b0d12`, акцент `#5b7cfa`, текст `#e6e8ef`.
