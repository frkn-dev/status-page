Privacy Policy

Last updated: June 26, 2026

What this extension does

FRKN Service Checker is a browser extension that helps you diagnose your internet connection: it checks the availability of popular websites and services, shows your external IP address, tries to detect VPN/proxy usage, checks FRKN server status, and measures connection speed. All checks are performed directly in the user's browser.

What data is collected

The extension does not collect, store, or transmit any personal data to our servers. The only information it obtains is public data required for the features to work:

  • External IP address, country, city, ISP, and ASN via public geolocation services (ipapi.co, ipwho.is, ipinfo.io).
  • FRKN node list and API status from api.frkn.org.
  • Results of HEAD/GET requests to the checked websites (availability status and latency only).
  • For speed tests, test files are downloaded from status.frkn.org and public mirrors.

Permissions

The extension requests access to <all_urls> because it checks the availability of many different websites directly from the user's browser. Without this permission, it is impossible to tell a real website response from an ISP block page.

Data storage

All check results are shown only inside the extension popup and are not stored outside the browser. The extension does not use third-party analytics, trackers, or advertising.

Contact

If you have any questions about this privacy policy, please contact us at support@frkn.org.
